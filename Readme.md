Binary Tree Common Parent
========================
  This is an example of traversing through a binary tree and returning the least common ancestor given the two nodes to search for. 
  
  Examples 
---------------
Given the following binary tree.

				a
			  /  \
			b     c
		   / \   / \
		 d   e  f   g
	           / 	 
	          i 
	          
####Tests	          

|    Node1  |   Node2   | LCA  |
|:---------:|:---------:|:----:|
|      d    |      e    |  b   |
|      f    |      g    |  c   |
|      d    |      g    |  a   |
|      i    |      c    |  c   |
